# Project Purpose

## Current goal
Understand how to find good stocks to invest in by researching investment methodologies, defining quantitative screening criteria, and building/documenting a stock finder tool.

## Success criteria
- Define a clear framework of fundamental and technical indicators for "good stocks" (e.g., valuation, profitability, growth, financial health).
- Identify and document reliable, free or cost-effective financial data APIs or sources (e.g., Yahoo Finance, Alpha Vantage, Finnhub).
- Implement a screening/analysis tool (CLI or web interface) that fetches stock data and ranks/filters companies according to the framework.
- Produce a clear, actionable guide/documentation summarizing the findings and investment methodology.
